Documentation

This section provides the guide for setting up, testing, and customizing your email templates.

TIP: How to Use FTP — Learn More ↗
Note
The most common issue in the installation process is to try to install the email templates as a zip package in Plugins > Add New > Upload Plugin or Appearance > Themes > Add New > Upload Theme. WooCommerce Email Templates is not a plugin or WordPress theme.

Setting Up Email Templates

  1. 1

    Connect to FTP

    Create a \woocommerce\emails\ folder structure in your active theme. For example,

    \[root]\wp-content\themes\[YourTheme]\woocommerce\emails\
  2. 2

    Upload Files via FTP

    Upload the content of the email template folder (email-header.php, email-footer-php, etc.) to the \woocommerce\emails\ folder. For example,

    \[root]\wp-content\themes\[YourTheme]\woocommerce\emails\email-header.php
    \[root]\wp-content\themes\[YourTheme]\woocommerce\emails\email-footer.php
    etc.
  3. 3

    Set Up the Template

    Go to WooCommerce > Settings > Emails and set the logo, footer text and colors.

Frequently Asked Questions

What do you mean by “one purchase for all your future WooCommerce based projects”?

At least four times per year, we come with new email template designs & bug-fix updates. We want to build a community around email templates and offer this community our other solutions.

What is included?

All standard transactional emails are included. If you are using a third-party plugin to send emails, then you will have to ask those developers how to set their emails.

Can I use all 20 styles?

Yes, but keep in mind that a website can use only one active email template at a time.

Is it a plugin?

No, it is a design. Think about email templates as a WooCommerce theme but for emails.

Do I need coding skills?

Basic customization, like changing the logo, text, and colors, can be done from your WooCommerce dashboard. For more advanced customization, you will need some basic HTML & CSS skills.

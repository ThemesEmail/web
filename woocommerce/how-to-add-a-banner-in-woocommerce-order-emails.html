How to Add a Banner in
WooCommerce Order Emails

Category: WooCommerce
  1. 1

    Upload your banner to your WordPress or CDN.

  2. 2

    Link the URL of the banner in HTML—for example,

    <img src="https://themes.email/images/banner.png" alt="Banner">
  3. 3

    Locate where you want to place the banner in the email-header.php or email-footer.php file and add the HTML you created in step 2.



    <table border="0" cellpadding="0" cellspacing="0" width="100%">
      <tr>
            <td align="left" valign="top">
 	      <img src="https://themes.email/images/banner.png" alt="Banner">
            </td>
      </tr>
</table>
  4. 4

    Upload the changed file via FTP to overwrite the old files.

    General

  • It is a plugin?
    Email Templates are generally HTML emails but intended for CMS systems. You can change design of your emails. They are not adding any new functionality to your CMS.
  • Do I need coding skills?
    Basic customization, like changing the logo, text, and colors, can be done from the admin panel of your system. For more advanced customization, you will need some basic HTML & CSS skills.
